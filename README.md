# Thoughts Platform

A web-based platform for reading, writing, and managing articles with bookmarking and history features.

## Features

- 📝 **Article Editor** - Create and edit articles with a rich text editor
- 📚 **Article Library** - Browse and read saved articles
- 🔖 **Bookmarks** - Save your favorite articles for quick access
- 📥 **Downloads** - Download articles for offline reading
- 👥 **Follows** - Follow topics or authors
- 📜 **History** - Keep track of articles you've read
- 🔐 **User Authentication** - Secure login system

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Create an account and start exploring!
